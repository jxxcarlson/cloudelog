# Log start date — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add an optional `start_date` to logs. When set before today, the backend backfills zero-quantity skip entries for `[start_date, today)` in the same transaction as the log insert. All entries (including backfilled skips) become editable inline in `LogView.elm`.

**Architecture:** New `logs.start_date DATE NOT NULL DEFAULT CURRENT_DATE` column. `createLogHandler` becomes a single `Serializable Write` transaction: insert log, then bulk-insert skip entries via existing `insertSkipFills`. Entry editing reuses the existing `PUT /api/entries/:id` endpoint; a new inline edit form in `LogView.elm` wires it up to the UI. Date strings are handled as ISO `YYYY-MM-DD` end-to-end; the `<input type="date">` element emits them natively.

**Tech Stack:** Haskell (Servant, Hasql, Aeson), PostgreSQL, Elm (Browser.application), dbmate for migrations, `test-api.sh` for end-to-end verification.

**Spec:** `docs/superpowers/specs/2026-04-19-log-start-date-design.md`

```haskell
-- POST /api/logs
-- In a single transaction: insert the log, and if startDate < today,
-- bulk-insert zero-quantity skip entries for [startDate, today).
createLogHandler :: AuthResult AuthUser -> CreateLogRequest -> AppM LogResponse
createLogHandler auth CreateLogRequest{..} = do
  uid <- requireUser auth
  validateUnit clrUnit
  validateName clrName
  pool  <- asks envDbPool
  today <- liftIO (utctDay <$> getCurrentTime)
  startDate <- case clrStartDate of
    Nothing -> pure today
    Just d  -> do
      when (d > today) $
        throwError $ appErrorToServantErr (BadRequest "start date cannot be in the future")
      pure d
  lid <- liftIO $ toText <$> UUID.nextRandom
  let desc     = fromMaybe "" clrDescription
      fillDays = datesToFill (Just (addDays (-1) startDate)) today
  fillIds <- liftIO $ generateUuids (length fillDays)
  result <- liftIO $ Pool.use pool $
    Tx.transaction Tx.Serializable Tx.Write $ do
      l <- Tx.statement
             (lid, uid, clrName, desc, normalizeUnit clrUnit, startDate)
             DbLog.insertLog
      unless (null fillDays) $
        Tx.statement
          (lid, V.fromList fillIds, V.fromList fillDays)
          DbEntry.insertSkipFills
      pure l
  case result of
    Left _  -> throwError $ appErrorToServantErr (Internal "database error")
    Right l -> pure (toLogResponse l)

generateUuids :: Int -> IO [Text]
generateUuids n
  | n <= 0    = pure []
  | otherwise = mapM (\_ -> toText <$> UUID.nextRandom) [1..n]
```
